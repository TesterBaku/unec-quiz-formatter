# Lessons

- No project-specific lessons captured yet.
- When a user indicates they want dependency isolation through a virtual environment, set up the Python venv before choosing a different runtime path.
- When tightening UI spacing, inspect adjacent container gaps and inherited heading or form margins instead of changing only the most obvious gap rule.
- When a panel sits beside taller content in a CSS grid, check for default grid stretching first; use `align-items: start` on the parent grid and `align-content: start` on internal grid containers before chasing phantom gap values.
- When making code changes to this app, automatically rebuild the distributables (`.exe` and shareable `.zip`) as part of the same task unless the user explicitly says not to.
