# Quiz Formatter With ChatGPT

Приложение распознаёт вопросы с вариантами ответов из PDF, собирает случайные викторины, проверяет результаты и при необходимости показывает объяснения ошибок на основе ключа ответов, встроенного в PDF.

## Установка

```powershell
uv python install 3.12
uv venv .venv --python 3.12
.\.venv\Scripts\python.exe -m ensurepip --upgrade
.\.venv\Scripts\python.exe -m pip install pypdf pytest
.\.venv\Scripts\python.exe -m pip install -e .
```

## Запуск в терминале

```powershell
.\.venv\Scripts\python.exe -m quiz_formatter
```

## Запуск веб-интерфейса

```powershell
.\.venv\Scripts\python.exe -m quiz_formatter --web --port 8000
```

После этого откройте `http://127.0.0.1:8000`.

## Сборка `.exe`

```powershell
powershell -ExecutionPolicy Bypass -File .\build_exe.ps1
```

После сборки исполняемый файл будет здесь:

```text
dist\quiz_formatter.exe
```

Запуск готового `.exe`:

```powershell
.\dist\quiz_formatter.exe
.\dist\quiz_formatter.exe --web --port 8000
```

## Как делиться `.exe`

- Можно передать только `dist\quiz_formatter.exe`, если у получателя уже есть PDF и он запустит программу в папке с этим PDF или передаст путь к файлу вручную.
- Удобнее всего положить `quiz_formatter.exe` рядом с нужным PDF и отправить их вместе.
- История попыток будет сохраняться рядом с `.exe` в папку `.quiz-cache`.
- Если Windows покажет предупреждение SmartScreen, это нормально для неподписанного локально собранного `.exe`.

## Полезные параметры

- `--count 10` задаёт количество случайных вопросов.
- `--seed 1` делает выбор вопросов и перемешивание ответов повторяемыми.
- `--ranges "1-50; 120-160; 200"` ограничивает викторину выбранными номерами вопросов.
- `--show-correct-from-start` включает режим изучения: правильные ответы показываются сразу.
- По умолчанию варианты ответов перемешиваются. Используйте `--no-shuffle-answers`, чтобы оставить порядок из PDF.
- `--parse-only --preview 5` показывает распознанные вопросы без запуска викторины.
- `--show-history` выводит последние сохранённые попытки из `.quiz-cache/history.jsonl`.
- `path\to\file.pdf` позволяет указать конкретный PDF вместо первого найденного в папке.

## Как это работает

- Парсер считывает нумерованные вопросы из PDF.
- Вариант, отмеченный символом `√`, считается правильным ответом.
- Приложение может выбирать вопросы только из указанных диапазонов, например `1-50; 120-160`.
- Приложение случайно выбирает вопросы и может перемешивать порядок вариантов ответа.
- В режиме изучения правильные ответы показываются сразу, без подсчёта результата.
- Результаты викторин сохраняются в `.quiz-cache/history.jsonl`.
- После прохождения викторины приложение показывает итоговый результат.
- Если есть ошибки, приложение предлагает показать объяснения и правильные ответы.
